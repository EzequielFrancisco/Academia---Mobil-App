# Academia – Educational Mobile App

Academia is an educational mobile application designed to help students learn through interactive games such as quizzes and memory challenges.

The app allows users to create and personalise questions based on their school subjects, transforming traditional study material into engaging learning experiences. By playing, students memorise concepts naturally, without the pressure of conventional study methods.

## Features
- Quiz-based learning with custom questions
- Memory games focused on long-term retention
- Import and export of quizzes for different users
- Supports learning across multiple subjects (e.g. medicine, philosophy, languages)

## Educational Approach
Academia is based on the idea of **learning through repetition and play**, helping users strengthen memory in a natural way — similar to how daily habits improve skills without conscious effort.

## Technology
- Python
- KivyMD
- Mobile-focused UI

## Purpose
This project was created for **educational and learning purposes**.
